# Zadatak
Parsirati JSON fajl formata poput navedenog korišćenjem *data binding* metoda:
```json
{
	"brojKartona":"12/12",
	"lekar":{
		"ime":"Petar",
		"prezime":"Petrovic"
	},
	"pacijent":{
		"ime":"Mika",
		"prezime":"Mikic"
	},
	"dijagnoze":[
		{"naziv":"prehlada", "sifra": "PO1"},
		{"naziv":"grip", "sifra":"GO2"}
	]
}
```
